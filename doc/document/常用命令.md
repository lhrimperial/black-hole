#### 常用命令

1. 查看日志

   - 根据关键字查看日志：
     `cat catlina.log | grep "test"`

   - 根据关键字查看后20行：
     `cat catlina.log | grep "test" -A 20`

   - 根据关键字查看前20行：
     `cat catlina.log | grep "test" -B 20`

   - 根据关键字查看前后20行：
     `cat -n catlina.log | grep "test" -C 20`

   - 查看日志前100行：
     `cat catlina.log | head -n 100`

   - 查看日志后100行：
     `cat -n catlina.log | tail -n 100`

   >特殊字符说明：
   >
   >- A 表示关键字之后，After
   >- B 表示关键字之前，Before
   >- C 表示关键字前后，Context
